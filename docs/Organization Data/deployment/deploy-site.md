# Site Deployment

How to push your CMS changes to the live FHC website.

## Deployment Process
Most changes appear immediately after clicking **Publish**. However, structural changes or system updates may require a manual trigger.

1.  Log into the deployment dashboard (e.g., Vercel, Netlify, or Jenkins).
2.  Trigger a **New Deployment**.
3.  Wait for the build to complete (usually 2-5 minutes).

## Verifying Deployment
- Visit [fhc.gov.et](https://fhc.gov.et).
- Perform a hard refresh (**Ctrl+F5**) to clear cache.
- Verify that your new content or changes are visible.

---

## Security Note
Keep your deployment credentials confidential. Only authorized technical staff should trigger deployments.
