# Contact & Message Management

Monitor messages from website visitors and update the organisation's contact details.

---

## Viewing Contact Messages

1. Go to **Content Manager** → **Contact Message**.
2. Browse the list of inquiries submitted via the website contact form.
3. Check the **Subject**, **Message**, and **Sender Info** columns.

::: tip
Contact messages are read-only. You cannot reply from here — use the sender's email address to respond directly.
:::

---

## Updating Contact Details

The headquarters address, main email, and phone numbers displayed in the website footer are managed in **Site Settings**:

1. Go to **Content Manager** → **Site Setting** (single type).
2. Update the **Contact Email**, **Contact Phone**, and **Address** fields.
3. Click **Save**, then **Publish**.

::: warning
There is only one Site Setting entry. Changes publish immediately to the live website.
:::
