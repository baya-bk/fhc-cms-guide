# Troubleshooting Common Errors

Solutions for frequent CMS issues.

## 1. Login Failed
- Ensure you are using the correct email and password.
- Check if you have an active internet connection.
- Contact the IT team to reset your password if forgotten.

## 2. Content Not Appearing on Website
- Ensure you clicked **Publish**, not just **Save**.
- Clear your browser cache or try an Incognito window.
- Check if the site is currently being redeployed.

## 3. Upload Error
- Check the file size (max 20MB).
- Ensure you are uploading a supported file type (JPG, PNG, PDF).
