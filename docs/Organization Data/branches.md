# Manage Branches

Manage FHC's regional branch offices and their contact details.

---

## Adding a Branch

1. Go to **Content Manager** → **Branch**.
2. Click **+ Create new entry**.
3. Enter the **Branch Name** and **Address**.
4. Add the **Phone** number and **Email** address.
5. Click **Save**, then **Publish**.
git remote add origin https://github.com/<your-username>/fhc-cms-guide.git
git branch -M main
git push -u origin main
---

## Field Reference

| Field | Translated? | Notes |
|-------|-------------|-------|
| **Name** | ✅ Translate | Branch name in each language |
| **Address** | ✅ Translate | Full address text |
| **Phone** | ❌ Shared | International format preferred (e.g., +251 11 xxx xxxx) |
| **Email** | ❌ Shared | Official branch email address |

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Verify phone and email before publishing | Publish unverified contact details |
| Translate the branch name and address | Leave Amharic/Oromo fields as English |
| Use international phone format (+251…) | Use local shorthand numbers only |
