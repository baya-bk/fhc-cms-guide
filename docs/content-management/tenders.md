# Tenders & Procurement Management

Manage bid invitations, procurement notices, and tender opening events from this section.

---

## Required Fields

::: warning These fields are mandatory
- **Title** — description of what is being tendered
- **Reference** — unique tender number in the format `FHC/YYYY/NNN` (e.g., `FHC/2026/001`)
- **Deadline** — date and time submissions close
:::

---

## Creating a Tender

1. Go to **Content Manager** → **Tender**.
2. Click **+ Create new entry**.
3. Enter the **Title**, **Reference**, and **Deadline**.
4. Fill in the **Description** with full tender details.
5. Upload any **Documents** (PDF only — see guidelines below).
6. Click **Save**, then **Publish**.

---

## Field Reference

| Field | Translated? | Notes |
|-------|-------------|-------|
| **Title** | ✅ Translate | Short description of the tender |
| **Description** | ✅ Translate | Full tender details |
| **Category** | ✅ Translate | construction / goods / services |
| **Reference** | ❌ Shared | Unique number — do not change after publishing |
| **Deadline** | ❌ Shared | Same closing time in all languages |
| **Documents** | ❌ Shared | PDF attachments (see guidelines below) |
| **Media Embed** | ❌ Shared | YouTube/Vimeo link for opening session video |
| **Is Livestream** | ❌ Shared | Toggle on for live opening ceremonies |
| **Livestream Scheduled** | ❌ Shared | Date/time of the live opening |

---

## Adding a Video or Live Broadcast

Tender openings can be broadcast live or linked to a recorded session:

**For a recorded video:**
1. Paste the full YouTube or Vimeo URL into the **Media Embed** field.

**For a live broadcast:**
1. Toggle **Is Livestream** to `ON`.
2. Set **Livestream Scheduled** to the opening ceremony date and time.

::: tip
The website will automatically display a "LIVE" indicator at the scheduled time.
:::

---

## Document Guidelines

::: warning File requirements
- Only upload **PDF** files
- Maximum file size: **20 MB**
- Filename must not contain special characters — use hyphens instead (e.g., `fhc-tender-2026-001-invitation.pdf`)
:::

**Steps to upload documents:**
1. Scroll to the **Documents** field.
2. Click **Add a media** → **Upload new** or **Browse from library**.
3. Select your PDF file(s).
4. Click **Finish**.

---

## Tender Lifecycle

| Status | When to use |
|--------|-------------|
| **Draft** | Being prepared — not visible on the website |
| **Published** | Live and open for bidding |
| **Closed** | Deadline has passed — filter by deadline date to identify these |
| **Archived** | Historical record — keep but no longer active |

::: tip
There is no separate "Closed" status. Once the deadline passes, the tender is automatically treated as inactive on the frontend. Always archive old tenders after results are announced.
:::

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Use the standard reference format `FHC/YYYY/NNN` | Use free-form reference numbers |
| Upload a clear PDF with all specifications | Upload Word or Excel files |
| Set the correct deadline date and time | Leave deadline blank |
| Archive tenders after award results | Delete old tenders |
