# Job Openings Management

Manage career positions and recruitment announcements on the FHC website.

---

## Required Fields

::: warning These fields are mandatory
- **Title** — the job position name
- **Deadline** — application closing date
:::

---

## Creating a Job Opening

1. Go to **Content Manager** → **Job Opening**.
2. Click **+ Create new entry**.
3. Enter the **Title**, **Department**, and **Deadline**.
4. Write the **Description** and **Requirements** using the rich text editor.
5. Select the **Type**.
6. Click **Save**, then **Publish**.

---

## Field Reference

| Field | Translated? | Notes |
|-------|-------------|-------|
| **Title** | ✅ Translate | Job position name |
| **Department** | ✅ Translate | Department or unit name |
| **Description** | ✅ Translate | Full job description (rich text) |
| **Requirements** | ✅ Translate | Qualifications and experience needed (rich text) |
| **Deadline** | ❌ Shared | Application closing date — same in all languages |
| **Type** | ❌ Shared | full_time / part_time / contract |

---

## Employment Types

| Value | Label |
|-------|-------|
| `full_time` | Full Time |
| `part_time` | Part Time |
| `contract` | Contract |

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Set an accurate application deadline | Leave the deadline blank |
| Write clear requirements in plain language | Use overly technical jargon |
| Translate all fields into Amharic and Oromo | Publish only the English version |
| Remove or archive the entry once the position is filled | Leave filled positions live on the website |
