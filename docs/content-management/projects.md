# Projects Management

Track and showcase FHC's housing construction and development projects.

---

## Required Fields

::: warning These fields are mandatory
- **Title** — the project name
- **Project Status** — current stage of the project
- **Featured Image** — the main project photo shown on the website
:::

---

## Creating a Project

1. Go to **Content Manager** → **Project**.
2. Click **+ Create new entry**.
3. Fill in the **Title** and select the **Project Status**.
4. Upload the **Featured Image**.
5. Fill in optional fields as available.
6. Click **Save**, then **Publish**.

---

## Field Reference

| Field | Translated? | Notes |
|-------|-------------|-------|
| **Title** | ✅ Translate | Project name in each language |
| **Project Status** | ❌ Shared | planned / in_progress / completed |
| **Units** | ❌ Shared | Number of housing units (integer) |
| **Budget** | ❌ Shared | Display string (e.g., "50M ETB") |
| **Progress %** | ❌ Shared | Completion percentage from 0 to 100 |
| **Completion Date** | ❌ Shared | Expected finish date |
| **Featured Image** | ❌ Shared | Main photo — shared across all languages |
| **Gallery Images** | ❌ Shared | Additional photos — shared across all languages |
| **Location** | ❌ Shared | Linked location entry |
| **Progress Reports** | ❌ Shared | Linked progress report entries |

---

## Updating Project Progress

To update an ongoing project:

1. Open the project entry.
2. Update the **Progress %** field.
3. Upload any new photos to **Gallery Images**.
4. Link any new entries in **Progress Reports**.
5. Click **Save**, then **Publish**.

---

## Gallery Image Tips

- Order images logically (site overview → close-up details → progress shots).
- Compress images before upload using [TinyPNG](https://tinypng.com).
- Use consistent aspect ratios (16:9 recommended).

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Always upload a featured image | Publish without a cover photo |
| Keep progress % accurate | Leave it at 0 once work has started |
| Translate the project title | Leave Amharic/Oromo title as English |
| Archive completed projects correctly | Delete old projects |
