# About Page Management

Update the text on the FHC About Us page from this section.

---

## Important: This is a Single-Type Page

::: warning One entry only
The About Page has **a single entry**. There is no list — editing this entry directly updates what appears on the public website. Always review your changes carefully before clicking Publish.
:::

---

## Editing the About Page

1. Go to **Content Manager** → **About Page**.
2. The single entry opens automatically — there is no list to choose from.
3. Edit the relevant fields (see table below).
4. Click **Save**, then **Publish**.

---

## Field Reference

| Field | Description |
|-------|-------------|
| **Hero Title** | The large heading shown at the top of the About page |
| **Hero Description** | Short introductory text under the hero title |
| **Mission Title** | Heading for the Mission section |
| **Mission Content** | The organization's mission statement |
| **Vision Title** | Heading for the Vision section |
| **Vision Content** | The organization's vision statement |
| **CEO Message** | CEO name, photo, and message text |
| **Values** | List of core values (title + description for each) |

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Review the page on the website after publishing | Publish without previewing |
| Keep mission and vision statements concise | Write lengthy, unfocused paragraphs |
| Use the CEO's official title and current photo | Use outdated photos or informal titles |
