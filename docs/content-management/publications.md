# Publications Management

Manage annual reports, institutional magazines, and official documents from this section.

---

## Required Fields

::: warning These fields are mandatory
- **Title** — the name of the publication
- **File** — the PDF document to be downloaded
- **Publication Type** — category of the document
- **Year** — the year this publication covers
:::

---

## Uploading a Publication

1. Go to **Content Manager** → **Publication**.
2. Click **+ Create new entry**.
3. Enter the **Title** and select the **Publication Type**.
4. Set the **Year** field.
5. Click on the **File** field → **Upload new** → select your PDF.
6. Click **Save**, then **Publish**.

---

## Document Guidelines

::: warning File requirements
- Only upload **PDF** files
- Maximum file size: **20 MB**
- Use clean, descriptive filenames with no special characters
:::

**Recommended naming format:**
```
fhc-annual-report-2025.pdf
fhc-magazine-q1-2026.pdf
fhc-procurement-plan-2026.pdf
```

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Use a clear, descriptive title | Use vague names like "Report Final v2" |
| Name the file with year and type | Use filenames with spaces or special characters (`'`, `#`, `&`) |
| Compress large PDFs before uploading | Upload files exceeding 20 MB |
| Set the correct Year field | Leave Year blank |
