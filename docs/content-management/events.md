# Events Management

Manage upcoming FHC events from this section.

---

## Required Fields

::: warning These fields are mandatory
- **Title** — the event name
- **Date** — the event date and time
:::

---

## Creating an Event

1. Go to **Content Manager** → **Event**.
2. Click **+ Create new entry**.
3. Fill in the **Title**, **Date**, **Location**, and **Description**.
4. Upload a **Cover Image** if available.
5. Click **Save**, then **Publish**.

---

## Field Reference

| Field | Translated? | Notes |
|-------|-------------|-------|
| **Title** | ✅ Translate | Event name in each language |
| **Description** | ✅ Translate | Full event details (rich text) |
| **Location** | ✅ Translate | Venue name or address |
| **Category** | ✅ Translate | Type of event (e.g., conference, workshop) |
| **Date** | ❌ Shared | Same date/time in all languages |
| **Image** | ❌ Shared | One image used in all languages |

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Set an accurate date and time | Leave the date field empty |
| Translate the location name for clarity | Use English location names in Amharic/Oromo versions |
| Use a 16:9 aspect-ratio cover image | Upload portrait-orientation images |
