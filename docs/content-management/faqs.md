# FAQs Management

Manage frequently asked questions displayed on the FHC website.

---

## Creating a FAQ

1. Go to **Content Manager** → **FAQ**.
2. Click **+ Create new entry**.
3. Enter the **Question** and write the **Answer** using the rich text editor.
4. Set the **Category** to group related questions.
5. Set the **Order** number to control which questions appear first.
6. Click **Save**, then **Publish**.

---

## Field Reference

| Field | Translated? | Notes |
|-------|-------------|-------|
| **Question** | ✅ Translate | The question visitors ask |
| **Answer** | ✅ Translate | Full answer — supports rich text (lists, links, bold) |
| **Category** | ✅ Translate | Groups questions on the page (e.g., Resident, Contractor) |
| **Order** | ❌ Shared | Lower number = appears first |

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Use clear, simple language | Write overly technical or jargon-heavy answers |
| Group questions with consistent categories | Mix categories inconsistently |
| Set the **Order** field so they appear logically | Leave all FAQs at Order 0 |
| Translate all three language versions | Publish only the English version |
