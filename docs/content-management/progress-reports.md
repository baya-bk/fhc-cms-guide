# Progress Reports

Provide regular progress updates for specific projects to keep stakeholders informed.

---

## Required Fields

::: warning These fields are mandatory
- **Project** — the linked project this report belongs to
- **Completion Percentage** — current progress (0–100)
:::

::: warning Create the project first
A progress report can only be linked to an existing project. Make sure the project entry has been created and published before adding a report.
:::

---

## Creating a Progress Report

1. Go to **Content Manager** → **Progress Report**.
2. Click **+ Create new entry**.
3. Select the related **Project** from the dropdown.
4. Set the **Completion Percentage**.
5. Describe the current work phase and any notes.
6. Click **Save**, then **Publish**.

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Link the report to the correct project | Leave the project relation empty |
| Use clear, professional language | Use informal abbreviations |
| Update the report when phase changes | Create duplicate reports for the same phase |
| Keep the completion percentage accurate and current | Set it to 100% before the project is actually done |
