# Dashboard Overview

After logging in, you land on the Content Manager dashboard — your central hub for managing all FHC website content.

---

## Sidebar Navigation

| Section | What it does |
|---------|--------------|
| **Content Manager** | Create, edit, and publish all content (News, Tenders, Projects, etc.) |
| **Media Library** | Upload and manage images, PDFs, and other files |
| **Settings** | Manage users, roles, languages, and system configuration *(admin only)* |

---

## Content List View

When you select a content type from the sidebar, you see a list of all entries with:

- **Title** – name of the entry
- **Status** – `Draft` (not live) or `Published` (visible on the website)
- **Locale** – which language version you are viewing (`en`, `am-ET`, `om-ET`)
- **Updated at** – when it was last modified

---

## Key Tips

- Use the **Search** bar at the top of the list to find a specific entry quickly.
- Use the **Locale filter** (language dropdown) to switch between English, Amharic, and Oromo.
- Click the **Status** column header to sort by published/draft.
- Entries marked **Draft** are **not visible** on the public website until published.
