# File Management

Beyond images, the Media Library also handles PDF documents and other files used in publications, tenders, and downloadable resources.

---

## Uploading Files

1. Click **Media Library** in the sidebar.
2. Click **+ Add new assets**.
3. Drag and drop your file, or click to browse.
4. Click **Finish**.

After uploading, copy the **File URL** from the asset detail panel to use it in tender or publication entries.

---

## File Requirements

::: warning
- **Only PDF files** should be used for publications, tenders, and legal documents
- **Maximum file size**: 20 MB
- Filenames must **not contain** spaces, Arabic characters, or special symbols (`'`, `#`, `&`, `(`, `)`)
:::

**Recommended naming format:**
```
fhc-tender-2026-001-bid-invitation.pdf
fhc-annual-report-2025.pdf
fhc-procurement-plan-q1-2026.pdf
```

---

## Replacing a File

If you need to update an existing file (e.g., a corrected tender document):

::: warning Replace, don't edit
Do **not** try to retype the filename or overwrite in place. Delete the old file and upload the new one to avoid caching issues.
:::

1. Find the old file in **Media Library**.
2. Click the three-dot menu → **Delete**.
3. Upload the new file with the same (or updated) filename.
4. Update any publication or tender entries that referenced the old file.

---

## Organising Files

- Create folders by year or type: `Tenders/2026`, `Publications/Annual Reports`.
- Keep folder names short and consistent.

---

## Best Practices

| ✅ Do | ❌ Don't |
|------|---------|
| Use descriptive, hyphenated filenames | Use filenames with spaces or special characters |
| Compress PDFs over 10 MB before uploading | Upload unnecessarily large files |
| Delete unused files to keep the library clean | Leave old or duplicate files in the library |
| Update all related entries after replacing a file | Leave broken file links in published content |
